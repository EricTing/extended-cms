eXtended CMS

calculated extended contact mode score provided the query and template protein-ligand structures

Installation

The easiest way is to use pip.

pip install xcms

To install locally

pip install xcms --user

Test

In command line, go to the uncharived xcms directory, in most cases it would be extended-cms. Then follow the commands below:

cd ./tests/
py.test ./tests/test_cli.py

Usage

To use it:

x-cms --help
